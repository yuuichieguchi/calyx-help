主要なキーボードショートカットの一覧です。
Settings から個別に変更できる項目もあります。

## グループ操作（Ctrl+Shift 系）

| 操作 | ショートカット |
|---|---|
| 次のグループ | `Ctrl+Shift+]` |
| 前のグループ | `Ctrl+Shift+[` |
| 新しいグループ | `Ctrl+Shift+N` |
| 現在のグループを閉じる | `Ctrl+Shift+W` |

## タブ操作（Cmd 系）

| 操作 | ショートカット |
|---|---|
| 新しいタブ | `Cmd+T` |
| タブを閉じる | `Cmd+W` |
| 番号でタブを切り替え | `Cmd+1` 〜 `Cmd+9` |
| 次のタブ | `Cmd+Shift+]` |
| 前のタブ | `Cmd+Shift+[` |

## スプリット操作

| 操作 | ショートカット |
|---|---|
| 右に分割 | `Cmd+D` |
| 下に分割 | `Cmd+Shift+D` |
| 隣のペインへフォーカス | `Cmd+Option+矢印キー` |

## 検索

| 操作 | ショートカット |
|---|---|
| スクロールバック検索 | `Cmd+F` |
| 次の一致 | `Cmd+G` |
| 前の一致 | `Cmd+Shift+G` |
| 検索バーを閉じる | `Escape` |

## 通知

| 操作 | ショートカット |
|---|---|
| 最新の未読通知タブへジャンプ | `Cmd+Shift+U` |

## グローバル

| 操作 | ショートカット |
|---|---|
| コマンドパレット | `Cmd+Shift+P` |
| コンポーズオーバーレイの開閉 | `Cmd+Shift+E` |

## コンポーズオーバーレイ内

| 操作 | ショートカット |
|---|---|
| ターミナルに送信 | `Enter` |
| 改行を入れる | `Shift+Enter` |
| オーバーレイを閉じる | `Escape` |
